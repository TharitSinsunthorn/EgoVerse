## Installation
Do NOT activate the `emimic` venv for any of these steps. All commands use openpi's own isolated uv environment.

From EgoVerse directory:
```
cd external/openpi
uv pip install -e .
uv pip install transformers==4.53.2
cp -r ./src/openpi/models_pytorch/transformers_replace/* .venv/lib/python3.11/site-packages/transformers/
```

## Getting pretrained checkpoint
From EgoVerse directory:
```
mkdir -p egomimic/algo/pi_checkpoints
gsutil -m cp -r gs://openpi-assets/checkpoints/pi05_base egomimic/algo/pi_checkpoints/
```

Convert JAX checkpoint into Pytorch (run from inside `external/openpi/`):
```
cd external/openpi
uv run examples/convert_jax_model_to_pytorch.py \
  --config_name pi05_aloha \
  --checkpoint_dir ../../egomimic/algo/pi_checkpoints/pi05_base \
  --output_path ../../egomimic/algo/pi_checkpoints/pi05_base_pytorch
```
